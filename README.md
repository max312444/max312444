## プロジェクト | Projects

- **ストレートネック測定 Web アプリケーション**  
  Web カメラ入力を用いたリアルタイム姿勢分析 Web アプリ（Vue.js・Node.js・Express）  
  → カメラ入力処理、姿勢データ算出、結果の可視化までを一貫して実装

- **AI 自律走行 RC カープロジェクト**  
  カメラ映像のみで走行判断を行う End-to-End 自律走行システム（Python・OpenCV・CNN）  
  → 学習用データ作成から推論モデル実装、走行検証までを経験

- **サッカー情報掲示板 Web アプリケーション**  
  外部 API を活用したニュース・リーグ順位・試合日程表示 Web サービス  
  → 複数 API のデータ取得・加工・画面表示を実装

- **Bannote 自習室予約サービス（バックエンド）**  
  学内向け自習室予約 Web サービスのバックエンド開発を担当  
  → 予約作成・キャンセル・時間帯管理などの API 設計および実装  
  → データ整合性を意識したロジック設計と DB 操作を経験
